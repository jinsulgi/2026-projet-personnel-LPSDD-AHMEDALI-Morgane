# 2026-projet-personnel-LPSDD-AHMEDALI-Morgane

Étude de l'anglicisation des paroles K-pop (2000–2022) à partir du corpus Genius.

## Notebooks
- `data_prep_kpop_lyrics.ipynb` — Préparation et filtrage du dataset Kaggle
- `data_analysis_kpop_lyrics.ipynb` — Analyse linguistique (CS_ratio, TTR)

## Données
